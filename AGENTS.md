# Projeto

Trabalho da disciplina de Computação Distribuída.

Objetivo:
Implementar uma simulação de rede P2P não estruturada para comparar algoritmos de busca.

Algoritmos obrigatórios:

- flooding
- informed_flooding
- random_walk
- informed_random_walk

Requisitos:

- Ler topologia a partir de YAML
- Validar conectividade da rede
- Validar min_neighbors
- Validar max_neighbors
- Validar ausência de self-loops
- Validar que todos os nós possuem recursos

Métricas obrigatórias:

- total_messages
- total_nodes_involved
- resource_found

Prioridades:

1. Código simples
2. Arquitetura orientada a objetos
3. Cobertura de testes
4. Reprodutibilidade dos experimentos

Sempre gerar código documentado e tipado.
Sempre criar testes unitários.
Sempre atualizar README quando novas funcionalidades forem implementadas.
